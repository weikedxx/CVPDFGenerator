# CV PDF Generator

En CV-generator bygget med React og Tailwind CSS. Applikasjonen lar deg redigere CV-en din og laste den ned som PDF.

## Funksjoner

- ✏️ **Redigerbar CV** - Rediger alle seksjoner direkte i nettleseren
- 📄 **PDF-eksport** - Last ned CV-en som en profesjonell PDF
- 🎨 **Moderne design** - Rent og profesjonelt utseende med Tailwind CSS
- 🇳🇴 **Norsk språk** - Fullstendig på norsk (bokmål)
- 📱 **Responsiv** - Fungerer på både desktop og mobil

## CV-seksjoner

- **Kontaktinformasjon** - Navn, fødselsår, telefon og e-post
- **Personlig oppsummering** - Kort beskrivelse av deg selv
- **Ferdigheter** - Liste over dine nøkkelferdigheter
- **Arbeidserfaring** - Tidligere stillinger med beskrivelser
- **Utdanning** - Utdanningsbakgrunn
- **Språk** - Språkferdigheter med nivå
- **Kurs og sertifiseringer** - Relevante kurs
- **Lenker** - LinkedIn, nettside, etc.

## Kom i gang

### Forutsetninger

- Node.js (versjon 18 eller høyere)
- npm eller yarn

### Installasjon

1. Installer avhengigheter:
```bash
npm install
```

2. Start utviklingsserveren:
```bash
npm run dev
```

3. Åpne nettleseren på `http://localhost:5173`

### Bygg for produksjon

```bash
npm run build
```

Dette oppretter en `dist`-mappe med produksjonsklare filer.

## Teknologier

- [React](https://react.dev/) - UI-bibliotek
- [Tailwind CSS](https://tailwindcss.com/) - Styling
- [Vite](https://vitejs.dev/) - Byggeverktøy
- [html2pdf.js](https://github.com/eKoopmans/html2pdf.js) - PDF-generering
- [Lucide React](https://lucide.dev/) - Ikoner
